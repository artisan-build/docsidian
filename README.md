# Docsidian

Docsidian is a package that publishes your Obsidian vault to a Laravel site. Here are our favorite use cases for it:

* Documentation Site
* Blog
* Knowledge Base
* Simply publish your vault

## Installation

`composer require artisan-build/docsidian`
